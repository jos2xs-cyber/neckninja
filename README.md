# Scale Shed

A guitar fretboard visualization tool for learning scales, chords, and triads.

## Features

- **Scales**: View pentatonic, major, minor, and other scales across the full neck or by CAGED position
- **Chords**: See chord voicings with customizable example keys
- **Triads**: Visualize major and minor triads across string groups with interval coloring (Root/3rd/5th)
- **Display options**: Toggle note names (ABC) or interval labels (R35)
- **Dark/Light mode**: Easy on the eyes for late night practice sessions

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

3. Open http://localhost:5173 in your browser

## Tech Stack

- React 19
- TypeScript
